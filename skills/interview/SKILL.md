# Interview Skill

## 개요

사용자와의 대화를 통해 프로젝트 요구사항을 수집하고 구체화합니다.
심층 질문을 통해 모호한 부분을 명확히 하고, 최종적으로 상세 스펙 문서를 작성합니다.

---

## 사용 시점

- `/interview` 명령어
- "인터뷰 시작", "요구사항 정리해줘"
- 새 프로젝트 시작 시
- 기능 기획이 필요할 때

---

## 실행 방법

### 1. 기존 컨텍스트 확인

```
확인 파일:
- ~/.claude/plans/*.md (최신 계획 파일)
- .claude/SPEC.md (프로젝트 명세)
- CLAUDE.md (프로젝트 규칙)
```

### 2. 영역별 심층 질문

AskUserQuestion 도구를 사용하여 다음 영역 질문:

#### 기술 구현 (Technical)
- 기술 스택 선택
- 아키텍처 결정
- 성능 요구사항
- 확장성 고려사항

#### UI/UX
- 사용자 흐름
- 주요 화면 구성
- 인터랙션 방식
- 접근성 요구사항

#### 비즈니스 로직
- 핵심 기능
- 에지 케이스
- 데이터 모델
- 비즈니스 규칙

#### 제약사항
- 기술적 제약
- 일정 제약
- 리소스 제약
- 호환성 요구사항

### 3. 스펙 문서 작성

수집된 정보를 바탕으로 상세 스펙 문서 작성:

```markdown
# Project Specification

## 프로젝트 개요
- 프로젝트명:
- 목적:
- 주요 기능:

## 기술 스택
| 구분 | 기술 |
|------|------|
| 언어 | |
| 프레임워크 | |
| 데이터베이스 | |

## 아키텍처
...

## 구현 계획
### Phase 1:
### Phase 2:
### Phase 3:

## 제약사항
...
```

## 출력

```markdown
## 인터뷰 완료

### 수집된 요구사항 요약

**프로젝트**: <프로젝트명>
**목적**: <한 줄 설명>

**주요 기능**:
1. <기능 1>
2. <기능 2>
3. <기능 3>

**기술 스택**: <선택된 스택>

### 작성된 문서

`/plan` 명령으로 구현 계획을 수립하세요.
```

## 질문 규칙

### 해야 할 것

1. **심층 질문**: 표면적이 아닌 근본적인 질문
   - Bad: "어떤 기능이 필요하세요?"
   - Good: "사용자가 이 기능을 통해 해결하려는 문제는 무엇인가요?"

2. **구체화 질문**: 모호한 답변 구체화
   - "빠른 성능" → "응답 시간 몇 초 이내를 목표로 하시나요?"

3. **확인 질문**: 이해한 내용 확인
   - "제가 이해한 바로는 ~인데, 맞나요?"

4. **대안 제시**: 선택지 제공
   - "A 방식과 B 방식이 있는데, 어떤 것이 더 적합할까요?"

### 하지 말아야 할 것

- 뻔한 질문 (이미 명확한 내용)
- 기술적 세부사항으로 압도
- 한 번에 너무 많은 질문
- 사용자 답변 무시

## 예시

### 입력
```
사용자: "블로그 플랫폼을 만들고 싶어요"
```

### 인터뷰 진행
```
AI: 블로그 플랫폼 개발을 시작하겠습니다. 몇 가지 질문드리겠습니다.

Q1. 이 블로그의 주요 사용자는 누구인가요?
   - [ ] 개인 블로거 (1인 사용)
   - [ ] 다중 사용자 (팀/회사)
   - [ ] 공개 플랫폼 (누구나 가입)

Q2. 핵심 기능 우선순위를 정해주세요:
   - [ ] 글 작성/편집
   - [ ] 댓글 시스템
   - [ ] 구독/알림
   - [ ] SEO 최적화
```

### 출력
```markdown
## 인터뷰 완료

### 수집된 요구사항 요약

**프로젝트**: Personal Blog Platform
**목적**: 개인 기술 블로그 운영을 위한 플랫폼

**주요 기능**:
1. 마크다운 기반 글 작성
2. 카테고리/태그 관리
3. 댓글 시스템 (GitHub OAuth)
4. SEO 최적화

**기술 스택**: Next.js + MDX + Tailwind CSS

### 작성된 문서
- `.claude/SPEC.md` - 상세 스펙 문서

### 다음 단계
`/plan` 명령으로 구현 계획을 수립하세요.
```
